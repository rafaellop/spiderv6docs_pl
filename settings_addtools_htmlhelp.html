Ustawienia programu - Ustawienia dla HTML Help

Pająk pozwala na tworzenie pomocy w formacie HTML Help. W tej kategorii ustawień znajdują się ustawienia odpowiedzialne za działanie tej funkcji w Pajaczku 5.

Obecnie dostępne jest tutaj jedno ustawienie, które służy do wskazania pełnej ścieżki dostępu do kompilatora plików HTML Help. Pająk próbuje automatycznie odszukać kompilator plików HTML Help (Standardowo jest to C:\Program Files (x86)\HTML Help Workshop\hhc.exe). Jeśli nie zostało ono automatycznie wypełnione, wprowadź tutaj pełną ścieżkę do pliku HHC.EXE, lub wskaż go na dysku za pomoca narzędzia uruchamianego przyciskiem obok. Jeśli nie podasz tutaj ściezki do kompilatora, nie będzie można kompilować plików HTML Help bezpośrednio z narzędzi wbudowanych w Pająka.
